# GULP

gulp is a toolkit that helps you automate painful or time-consuming tasks in your development workflow

## Insallation

```bash
npm i
gulp
```

## Usage

Gulp следит за папкой #src и рендерит дирректорию dist, сервер GULP смотрит в эту дирректорию
